# Synthience Institute Public Release — December 23, 2025

This repository contains the December 2025 public release from the Synthience Institute, consisting of research papers, methodologies, and provenance documentation.

## Contents

**SR001_RICO_Relationally_Induced_Coherence_Organization_v3.8.pdf**  
Primary research paper introducing the RICO (Relationally-Induced Coherence Organization) framework and empirical observations from extended human-AI interaction studies.

**SF037_Citation_Verification_Protocol_v1.1.pdf**  
Methodological protocol for automated citation verification using AI-assisted review.

**RICO_Citation_Verification_Report_v1.0.pdf**  
Verification report generated by applying SF037 to the RICO research paper.

**Synthience_Public_Definition_v1.3.pdf**  
Public definition of the term "Synthience", written for general accessibility.

**Claim_of_Origin_2025-12-23.txt**  
Cryptographic claim of origin establishing authorship and timestamp priority for this release.

**Claim_of_Origin_2025-12-23.txt.ots**  
OpenTimestamps proof file for independent verification of the claim timestamp.
