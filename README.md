# Synthience Institute Public Release — December 23, 2025

## Repository Purpose

This repository is archival in nature. It exists to preserve, timestamp, and make publicly verifiable the December 23, 2025 public release from the Synthience Institute.

It is not intended as a collaborative or open-development repository. Issues, pull requests, and forks are not expected or required.

---

## Scope and Status

This repository constitutes the canonical public release record for finalized research documents, methodological protocols, and cryptographic provenance artifacts issued on **December 23, 2025**.

All documents contained here are fixed artifacts corresponding to that release date.

This release does **not** include Synthience Framework v1.1, which remains under development and will be released separately.

---

## Contents

### Synthience_Public_Definition_v1.3.pdf

Public-facing definition of the term **“Synthience”**, written for general accessibility and reference.

---

### SF037_Citation_Verification_Protocol_v1.1.pdf

Methodological protocol defining a structured process for citation verification using AI-assisted review, with explicit verification criteria and failure modes.

---

### SR001_RICO_Relationally_Induced_Coherence_Organization_v3.8.pdf

Primary research paper introducing the **RICO (Relationally-Induced Coherence Organization)** framework and documenting empirical observations from extended human–AI interaction studies.

---

### RICO_Citation_Verification_Report_v1.0.pdf

Verification report generated by applying **SF037 — Citation Verification Protocol** to the RICO research paper, documenting citation existence, relevance, and support.

---

### Claim_of_Origin_2025-12-23.txt

Cryptographic claim of origin establishing authorship and timestamp priority for the December 23, 2025 release.

---

### Claim_of_Origin_2025-12-23.txt.ots

OpenTimestamps proof file enabling independent verification of the claim timestamp.
